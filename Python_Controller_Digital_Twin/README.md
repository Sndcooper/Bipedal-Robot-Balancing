# Python Controller & Digital Twin

This folder houses the higher-level logic, inverse kinematics, tuning scripts, and the digital twin of the bipedal robot.

## Files Description

* **`bipedal_digital_twin_controller.ipynb`**: A Jupyter Notebook (originally named `ax12_controller.ipynb`) acting as a digital twin & controller. Includes Python scripts used for debugging, performing Inverse Kinematics calculations on the legs, sending tuning parameters via Serial to the STM32 board, and visualizing the robot's physical state.
